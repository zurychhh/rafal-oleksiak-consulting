## üõ†Ô∏è UPLOAD TO NOTION - STEP-BY-STEP

### Upload Workspace Icon:
1. Open Notion
2. Click workspace name (top-left corner)
3. Settings & Members ‚Üí Settings
4. Click workspace icon area
5. Click "Upload"
6. Select: `oleksiak-logo-compact.png`
7. Click "Save"

### Upload Custom Emojis (Repeat 15x):
1. Edit any page in Notion
2. Type ":" to open emoji picker
3. Click "Upload" icon (bottom-right of picker)
4. Select emoji file (e.g., `emoji-done-oleksiak.png`)
5. ‚úÖ CHECK: "Add to workspace Emoji library"
6. Enter emoji name (e.g., "done")
7. Click "Save"
8. **Repeat for all 15 emojis**

### Add Page Icons:
1. Hover over page title area
2. Click "Add icon"
3. Click "Upload" tab
4. Select appropriate icon file
5. Click "Save"

### Add Cover Images:
1. Hover over top of page
2. Click "Add cover"
3. Click "Upload"
4. Select cover file
5. Adjust position if needed
